# Key Features

AWO Platform delivers a comprehensive suite of financial services designed specifically for African women, combining traditional savings groups with modern investment opportunities and personalized financial coaching.

## Core Platform Features

### 1. DIVA Scoring System + RTSM Assessment

<Info>
AWO's proprietary dual-engine scoring system that combines financial health assessment (DIVA) with risk tolerance evaluation (RTSM) for intelligent portfolio matching.
</Info>

**DIVA Score Components:**
- **Discipline (30%)**: Payment consistency, overdraft avoidance, savings regularity
- **Income (25%)**: Stability, growth potential, diversification
- **Velocity (25%)**: Cash flow timing and efficiency
- **Assets (20%)**: Savings rate, investment balance, net worth growth

**RTSM Integration:**
- Questionnaire-based risk assessment
- Behavioral pattern analysis
- Investment timeline evaluation
- Portfolio preference matching
