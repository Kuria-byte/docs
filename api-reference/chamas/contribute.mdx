# Make Contribution

Make financial contributions to your Chama groups, supporting the traditional African practice of collective savings that has empowered women for generations. Regular contributions build community wealth, improve individual DIVA scores, and strengthen social bonds within the group.

<Info>
Consistent Chama contributions demonstrate financial discipline and community commitment - core values that significantly improve your DIVA score and unlock better financial opportunities.
</Info>

## Authentication

This endpoint requires a valid JWT token with `MAKE_CONTRIBUTIONS` permission. Users can only contribute to Chamas they are members of.

```bash
Authorization: Bearer <access_token>
```

## Request

### Endpoint
```
POST /chamas/{chamaId}/contributions
```

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `chamaId` | string | Yes | Unique identifier for the Chama group |

### Request Body

```json
{
  "amount": 1500.00,
  "contribution_type": "regular",
  "payment_method": "mobile_money",
  "payment_provider": "m_pesa",
  "payment_details": {
    "phone_number": "+27821234567",
    "reference": "Ubuntu-June-2025"
  },
  "contribution_period": "2025-06",
  "notes": "June contribution - Ubuntu Women's Group",
  "auto_contribute": false,
  "late_fee_accepted": true
}
```

### Request Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `amount` | number | Yes | Contribution amount (positive decimal) |
| `contribution_type` | string | Yes | Type of contribution: `regular`, `catch_up`, `emergency_fund`, `penalty`, `voluntary` |
| `payment_method` | string | Yes | Payment method: `mobile_money`, `bank_transfer`, `card`, `crypto` |
| `payment_provider` | string | Yes | Provider: `m_pesa`, `ecocash`, `fnb`, `standard_bank`, etc. |
| `payment_details` | object | Yes | Provider-specific payment information |
| `contribution_period` | string | No | Target period (YYYY-MM format) |
| `notes` | string | No | Optional contribution notes (max 200 chars) |
| `auto_contribute` | boolean | No | Set up automatic future contributions |
| `late_fee_accepted` | boolean | No | Accept late fees if contribution is overdue |

### Contribution Types

| Type | Description | Amount Rules | Impact |
|------|-------------|--------------|--------|
| `regular` | Monthly scheduled contribution | Fixed amount as per Chama rules | High DIVA score boost |
| `catch_up` | Make up for missed contributions | Can be multiple periods | Restores DIVA score |
| `emergency_fund` | Contribute to group emergency reserves | Any amount above minimum | Community trust bonus |
| `penalty` | Late fees or disciplinary payments | As determined by group rules | Maintains good standing |
| `voluntary` | Extra contribution beyond requirements | Any amount | Extra DIVA score boost |

## DIVA Score Impact

<Note>
Chama contributions have the highest impact on your DIVA score:
- **Regular contributions**: +5-10 points per month
- **Consistency bonus**: +20 points for 6+ consecutive months
- **Community participation**: +15 points for active group involvement
- **Early payments**: +3 bonus points for contributing before deadline
- **Voluntary contributions**: +5-15 additional points based on amount
</Note>

## Cultural Context

Chama contributions represent more than financial transactions - they embody the African principle of Ubuntu: "I am because we are." Each contribution strengthens the collective financial power of the group and demonstrates commitment to community upliftment.

Traditional Chama practices that AWO preserves:
- **Transparency**: All members see contribution records
- **Accountability**: Group members encourage consistent participation  
- **Celebration**: Successful contributions are acknowledged by the community
- **Support**: Members help each other meet contribution obligations

### Common Error Codes

| Code | Status | Description | Solution |
|------|--------|-------------|----------|
| `CHAMA_NOT_FOUND` | 404 | Chama does not exist | Verify Chama ID |
| `NOT_A_MEMBER` | 403 | User is not a Chama member | Join Chama first |
| `INSUFFICIENT_FUNDS` | 400 | Not enough balance for contribution | Add funds or use different payment method |
| `CONTRIBUTION_ALREADY_MADE` | 409 | Contribution for this period already exists | Check contribution history |
| `LATE_FEE_REQUIRED` | 400 | Late contribution requires fee acceptance | Accept late fees or contact admin |
| `PAYMENT_METHOD_UNAVAILABLE` | 400 | Selected payment method not available | Use alternative payment method |
| `AMOUNT_BELOW_MINIMUM` | 400 | Contribution below required minimum | Increase contribution amount |
| `AMOUNT_ABOVE_MAXIMUM` | 400 | Contribution exceeds maximum allowed | Reduce amount or split contributions |
| `CHAMA_SUSPENDED` | 403 | Chama is temporarily suspended | Contact support |
| `MEMBER_SUSPENDED` | 403 | Member account is suspended | Resolve account issues |

```javascript JavaScript/TypeScript
import { AWOClient } from '@awo-platform/js-sdk';

// Make a regular monthly contribution
async function makeRegularContribution(chamaId, amount, paymentDetails) {
  try {

      console.log('🏆 Achievement unlocked: 6-month consistency streak!');
    }
    
    return contribution;
  } catch (error) {
    console.error('❌ Contribution failed:', error.message);

      console.log(`💡 Shortfall: ${error.details.currency} ${error.details.shortfall}`);

      console.log('⏰ This contribution is late and requires a fee');
      // Retry with late_fee_accepted: true
      return makeRegularContribution(chamaId, amount, paymentDetails);
    }
    
    throw error;
  }
}

      totalChamas: userChamas.length,
      upToDate: 0,

          summary.overdue++;

    console.log(`💰 Total contributed: ZAR ${summary.totalContributed.toLocaleString()}`);
    console.log(`📈 Total DIVA boost: +${summary.estimatedDivaBoost} points`);
    
    return summary;
  } catch (error) {
    console.error('❌ Failed to get contribution summary:', error.message);
    throw error;
  }
```

```python Python
import requests

from typing import Dict, List, Optional

class AWOChamaContributions:
    def __init__(self, api_key: str, base_url: str = "https://api.awo-platform.com/v1"):
        self.api_key = api_key
        self.base_url = base_url
        self.headers = {
            'Authorization': f'Bearer {api_key}',
            'Content-Type': 'application/json'
        }
    
    def make_contribution(self, chama_id: str, contribution_data: Dict) -> Dict:
        """Make a contribution to a Chama"""
        response = requests.post(
            f"{self.base_url}/chamas/{chama_id}/contributions",
            headers=self.headers,
            json=contribution_data
        )
        
        if response.status_code == 201:
            return response.json()['data']
        else:
            error_data = response.json()
            raise Exception(f"Contribution failed: {error_data['error']['message']}")
    
    def make_regular_contribution(self, chama_id: str, amount: float, 
                                payment_method: str, payment_details: Dict,
                                auto_setup: bool = False) -> Dict:
        """Make a regular monthly contribution"""
        current_period = datetime.now().strftime('%Y-%m')

            'amount': amount,

            'payment_method': payment_method,

            'auto_contribute': auto_setup,
            'late_fee_accepted': True,
            'notes': f'Regular contribution for {current_period}'

        try:
            result = self.make_contribution(chama_id, contribution_data)

            print(f"💰 Total: {result['contribution_details']['currency']} {result['contribution_details']['amount']}")
```

```php PHP
<?php

class AWOChamaContributions {
    private $apiKey;
    private $baseUrl;
    
    public function __construct($apiKey, $baseUrl = 'https://api.awo-platform.com/v1') {
        $this->apiKey = $apiKey;
        $this->baseUrl = $baseUrl;
    }
    
    public function makeContribution($chamaId, $contributionData) {
        $url = $this->baseUrl . '/chamas/' . $chamaId . '/contributions';
        
        $context = stream_context_create([
            'http' => [
                'header' => [
                    'Authorization: Bearer ' . $this->apiKey,
                    'Content-Type: application/json',
                    'Accept: application/json'
                ],
                'method' => 'POST',
                'content' => json_encode($contributionData)
            ]
        ]);
        
        $response = file_get_contents($url, false, $context);
        
        if ($response === false) {
            throw new Exception('Failed to make contribution');
        }
        
        $data = json_decode($response, true);
        
        if (!$data['success']) {
            throw new Exception('Contribution failed: ' . $data['error']['message']);
        }
        
        return $data['data'];
    }
    
    public function makeRegularContribution($chamaId, $amount, $paymentDetails, $autoSetup = false) {
        $currentPeriod = date('Y-m');
        
        $contributionData = [
            'amount' => $amount,
            'contribution_type' => 'regular',
            'payment_method' => $paymentDetails['method'],
            'payment_provider' => $paymentDetails['provider'],
            'payment_details' => $paymentDetails['details'],
            'contribution_period' => $currentPeriod,
            'auto_contribute' => $autoSetup,
            'late_fee_accepted' => true,
            'notes' => "Regular contribution for {$currentPeriod}"
        ];
        
        try {
            $result = $this->makeContribution($chamaId, $contributionData);
            
            echo "✅ Regular contribution successful!\n";
            echo "💰 Amount: {$result['contribution_details']['currency']} {$result['contribution_details']['amount']}\n";
            echo "📈 DIVA boost: +{$result['diva_score_impact']['total_points_earned']} points\n";
            echo "🎯 New score: {$result['diva_score_impact']['new_estimated_score']}\n";
            
            if ($result['chama_impact']['contribution_streak'] >= 6) {
                echo "🏆 Achievement: 6-month consistency streak!\n";
            }
            
            return $result;
            
        } catch (Exception $e) {
            echo "❌ Regular contribution failed: " . $e->getMessage() . "\n";
            throw $e;
        }
    }
    
    public function makeCatchUpContribution($chamaId, $missedPeriods, $amountPerPeriod, $paymentDetails) {
        $totalAmount = count($missedPeriods) * $amountPerPeriod;
        
        $contributionData = [
            'amount' => $totalAmount,
            'contribution_type' => 'catch_up',
            'payment_method' => $paymentDetails['method'],
            'payment_provider' => $paymentDetails['provider'],
            'payment_details' => $paymentDetails['details'],
            'late_fee_accepted' => true,
            'notes' => 'Catch-up for periods: ' . implode(', ', $missedPeriods)
        ];
        
        try {
            $result = $this->makeContribution($chamaId, $contributionData);
            
            echo "✅ Catch-up contribution successful!\n";
            echo "📅 Periods covered: " . implode(', ', $missedPeriods) . "\n";
            echo "💰 Total: {$result['contribution_details']['currency']} {$result['contribution_details']['amount']}\n";
            echo "📈 DIVA restoration: +{$result['diva_score_impact']['total_points_earned']} points\n";
            
            return $result;
            
        } catch (Exception $e) {
            echo "❌ Catch-up contribution failed: " . $e->getMessage() . "\n";
            throw $e;
        }
    }
    
    public function calculateContributionImpact($userId, $chamaId, $amount, $contributionType = 'regular') {
        // Calculate estimated DIVA score impact
        $basePoints = floor($amount / 100); // Base points from amount
        $typeMultiplier = [
            'regular' => 1.0,
            'catch_up' => 0.8,
            'voluntary' => 1.2,
            'emergency_fund' => 1.1
        ];
        
        $disciplinePoints = $basePoints * ($typeMultiplier[$contributionType] ?? 1.0);
        $assetsPoints = floor($amount / 200);
        $communityPoints = 5; // Base community participation
        
        // Consistency bonus (simulated)
        $consistencyBonus = ($contributionType === 'regular') ? 3 : 0;
        
        return [
            'estimated_diva_boost' => [
                'discipline_points' => $disciplinePoints,
                'assets_points' => $assetsPoints,
                'community_points' => $communityPoints,
                'consistency_bonus' => $consistencyBonus,
                'total_points' => $disciplinePoints + $assetsPoints + $communityPoints + $consistencyBonus
            ],
            'financial_impact' => [
                'contribution_amount' => $amount,
                'estimated_fees' => $amount * 0.025, // 2.5% estimated fees
                'net_savings_increase' => $amount * 0.975
            ],
            'community_impact' => [
                'group_fund_contribution' => $amount,
                'member_trust_boost' => $contributionType === 'voluntary' ? 'high' : 'medium',
                'leadership_points' => $contributionType === 'emergency_fund' ? 5 : 0
            ]
        ];
    }
```
