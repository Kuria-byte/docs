# Create Savings Goal

Create and manage personalized savings goals that align with your financial dreams and priorities. AWO's smart goal-setting system provides culturally relevant recommendations, automatic progress tracking, and DIVA score integration to accelerate your journey to financial independence.

<Info>
Structured savings goals are proven to increase achievement rates by 42%. AWO's goal-setting system combines traditional African savings wisdom with modern financial planning to maximize your success.
</Info>

## Authentication

This endpoint requires a valid JWT token with `CREATE_SAVINGS_GOALS` permission.

```bash
Authorization: Bearer <access_token>
```

## DIVA Score Impact

<Note>
Savings goals significantly boost your DIVA score:
- **Goal Creation**: +10 points immediately for planning ahead
- **Monthly Contributions**: +8-15 points per month based on consistency
- **Milestone Achievement**: +5-20 points per milestone reached
- **Goal Completion**: +30-100 points based on goal size and timeline
- **Emergency Fund Bonus**: Extra +25 points for completing emergency fund
- **Multiple Goal Management**: +5 points monthly for managing 2+ active goals
</Note>

### DIVA Score Calculation for Goals

```javascript
// DIVA score boost formula for savings goals
const calculateGoalDivaBoost = (goal) => {
  const basePoints = {
    creation: 10,
    monthlyContribution: Math.min(15, goal.monthlyContribution / 500),
    priorityMultiplier: {
      low: 0.8,
      medium: 1.0,
      high: 1.2,
      critical: 1.5
    },
    categoryBonus: {
      emergency_fund: 5,
      education: 4,
      housing: 4,
      retirement: 6,
      healthcare: 3,
      business_startup: 3
    }
  };
  
  const monthlyBoost = basePoints.monthlyContribution * 
                      basePoints.priorityMultiplier[goal.priority] +
                      (basePoints.categoryBonus[goal.category] || 0);
  
  return {
    creation: basePoints.creation,
    monthly: monthlyBoost,
    milestoneAverage: monthlyBoost * 0.5,
    completionBonus: goal.targetAmount / 1000 * 2 // 2 points per 1000 currency units
  };
};
```

## Cultural Context

Savings goals in African culture represent more than personal financial planning - they embody the principle of **"Ikusasa Lethu"** (Our Future) where individual financial stability strengthens the entire community.

### Traditional African Savings Wisdom Integrated:

- **"Save in Good Times for Bad Times"**: Emergency funds reflect traditional wisdom about preparing for uncertainty
- **Community Accountability**: Sharing goals with trusted community members increases success rates
- **Celebration of Milestones**: African cultures celebrate progress, not just completion
- **Family-First Approach**: Goals often consider extended family needs and responsibilities
- **Incremental Progress**: "Little by little makes a bundle" - consistent small contributions build wealth

### Common Error Codes

| Code | Status | Description | Solution |
|------|--------|-------------|----------|
| `INVALID_TARGET_AMOUNT` | 400 | Target amount below minimum or above maximum | Use amount between ZAR 100 and ZAR 10,000,000 |
| `INVALID_TARGET_DATE` | 400 | Target date in past or too far in future | Set date between 30 days and 50 years from now |
| `INSUFFICIENT_MONTHLY_CONTRIBUTION` | 400 | Monthly contribution too low to reach goal | Increase contribution amount or extend timeline |
| `EXCESSIVE_MONTHLY_CONTRIBUTION` | 400 | Monthly contribution exceeds 80% of income | Reduce contribution or verify income information |
| `GOAL_LIMIT_EXCEEDED` | 400 | User has reached maximum number of active goals | Complete or archive existing goals first |
| `INVALID_CATEGORY` | 400 | Goal category not recognized | Use valid category from supported list |
| `DUPLICATE_GOAL_NAME` | 409 | Goal name already exists for user | Choose different name or update existing goal |
| `ACCOUNT_NOT_FOUND` | 404 | Source account for automatic transfers not found | Link valid bank account or use different payment method |
| `INSUFFICIENT_PERMISSIONS` | 403 | User lacks permission to create savings goals | Complete KYC verification or account setup |

    def calculate_optimal_timeline(self, target_amount: float, 
                                 available_monthly: float, max_months: int = 60) -> Dict:
        """Calculate optimal timeline for a savings goal"""
        if available_monthly <= 0:
            raise ValueError("Monthly contribution must be positive")
        
        basic_months = target_amount / available_monthly
        
        # Add buffer for realistic planning
        realistic_months = min(basic_months * 1.1, max_months)
        
        # Calculate with different contribution scenarios
        scenarios = {
            'conservative': {
                'monthly': available_monthly * 0.8,
                'months': target_amount / (available_monthly * 0.8)
            },
            'moderate': {
                'monthly': available_monthly,
                'months': basic_months
            },
            'aggressive': {
                'monthly': available_monthly * 1.2,
                'months': target_amount / (available_monthly * 1.2)
            }
        }
        
        return {
            'target_amount': target_amount,
            'scenarios': scenarios,
            'recommended_scenario': 'moderate',
            'recommended_months': int(realistic_months),
            'recommended_monthly': target_amount / realistic_months,
            'tips': [
                f"Start with the moderate scenario: ZAR {available_monthly:,.2f}/month",
                f"Complete in {int(realistic_months)} months",
                "Consider boosting contributions during good months",
                "Use Chama payouts to accelerate progress"
            ]
        }
