# Update Savings Goal

Modify and optimize your existing savings goals as your life circumstances, income, or priorities change. AWO's flexible goal management helps you stay on track toward financial success while adapting to real-life changes.

<Info>
Life changes, and your financial goals should adapt too. Regular goal updates increase achievement rates by 35% and maintain motivation through changing circumstances.
</Info>

## Authentication

This endpoint requires a valid JWT token with `UPDATE_SAVINGS_GOALS` permission. Users can only update their own goals.

```bash
Authorization: Bearer <access_token>
```

## Request

### Endpoint
```
PUT /savings/goals/{goalId}
```

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `goalId` | string | Yes | Unique identifier for the savings goal |

### Update Reasons

| Reason | Description | Common Changes |
|--------|-------------|----------------|
| `income_change` | Salary increase/decrease, new job | Monthly contribution, timeline |
| `family_growth` | New baby, marriage, dependents | Target amount, priority |
| `emergency` | Unexpected expenses, setbacks | Timeline extension, amount reduction |
| `opportunity` | Bonus, inheritance, windfall | Accelerated timeline, increased target |
| `life_stage` | Career change, retirement planning | Category change, strategy overhaul |
| `market_conditions` | Inflation, economic changes | Amount adjustment, timeline modification |
| `goal_refinement` | Better understanding of needs | All aspects potential update |

## DIVA Score Impact

<Note>
Goal updates show financial adaptability and responsibility:
- **Proactive Updates**: +5-10 points for updating before problems arise
- **Increased Commitment**: +8-15 points for raising targets or contributions
- **Life Event Adaptation**: +5-12 points for smart adjustments to life changes
- **Timeline Management**: +3-8 points for realistic timeline adjustments
- **Family Responsibility**: +5-10 points for family-focused goal updates
</Note>

### Update Impact Formula

```javascript
// DIVA score calculation for goal updates
const calculateUpdateImpact = (update) => {
  const basePoints = {
    proactiveUpdate: 5,
    increasedCommitment: update.targetIncrease > 0 ? Math.min(15, update.targetIncrease / 2000) : 0,
    timelineRealism: update.timelineAdjustment === 'realistic' ? 8 : 0,
    familyFocus: update.familyImpact ? 10 : 0,
    smartFeatureAdoption: update.newSmartFeatures * 2
  };
  
  const multipliers = {
    frequency: update.isFirstUpdate ? 1.0 : 0.8,
    priority: {
      critical: 1.2,
      high: 1.0,
      medium: 0.8,
      low: 0.6
    }
  };
  
  return Object.values(basePoints).reduce((sum, points) => sum + points, 0) * 
         multipliers.frequency * 
         (multipliers.priority[update.priority] || 1.0);
};
```

## Cultural Context

Goal updates in African culture reflect the principle of **"Ukuphila Ngokwenene"** (living authentically) - adapting plans to life's realities while maintaining commitment to family and community welfare.

### Traditional Wisdom in Goal Updates:
- **"Bend but don't break"**: Adjust goals rather than abandon them
- **Family first**: Always consider impact on extended family
- **Community accountability**: Seek wisdom from trusted community members
- **Seasonal thinking**: Align updates with natural cycles and seasons
- **Ubuntu approach**: Consider how personal changes affect the whole community

## Error Handling

### Common Error Codes

| Code | Status | Description | Solution |
|------|--------|-------------|----------|
| `GOAL_NOT_FOUND` | 404 | Goal does not exist | Verify goal ID |
| `GOAL_ALREADY_COMPLETED` | 409 | Cannot update completed goal | Archive or create new goal |
| `INVALID_TARGET_REDUCTION` | 400 | Cannot reduce target below current amount | Adjust target or create new goal |
| `EXCESSIVE_TIMELINE_EXTENSION` | 400 | Timeline extension too large | Use reasonable timeline extension |
| `INSUFFICIENT_CONTRIBUTION` | 400 | New contribution too low for timeline | Increase contribution or extend timeline |
| `CONFLICTING_UPDATES` | 400 | Update parameters conflict with each other | Review and resolve conflicting parameters |
| `UPDATE_FREQUENCY_EXCEEDED` | 429 | Too many updates in short period | Wait before making additional updates |
| `ACCOUNT_CHANGE_RESTRICTED` | 403 | Cannot change source account | Verify account ownership |

// Smart goal optimization based on progress analysis
async function optimizeGoalBasedOnProgress(goalId) {
  try {
    const currentGoal = await awo.savings.getGoal(goalId);
    const progressAnalysis = await awo.savings.analyzeProgress(goalId);
    
    let optimizations = {};
    
    // If consistently saving more than planned, accelerate timeline
    if (progressAnalysis.averageMonthlyActual > currentGoal.savings_plan.monthly_contribution * 1.2) {
      const acceleratedContribution = Math.round(progressAnalysis.averageMonthlyActual);
      optimizations.savings_strategy_updates = {
        monthly_contribution: acceleratedContribution
      };
      
      console.log(`üí™ Detected higher actual contributions: ${acceleratedContribution} vs planned ${currentGoal.savings_plan.monthly_contribution}`);
    }
    
    // If behind schedule, suggest realistic adjustments
    if (progressAnalysis.completionProbability < 70) {
      const currentDate = new Date();
      const targetDate = new Date(currentGoal.goal_details.target_date);
      const monthsRemaining = (targetDate.getTime() - currentDate.getTime()) / (1000 * 60 * 60 * 24 * 30);
      const amountRemaining = currentGoal.goal_details.target_amount - currentGoal.progress_tracking.current_amount;
      const requiredMonthly = amountRemaining / monthsRemaining;
      
      if (requiredMonthly > currentGoal.savings_plan.monthly_contribution * 1.5) {
        // Extend timeline rather than drastically increase contributions
        const realisticMonthly = currentGoal.savings_plan.monthly_contribution * 1.2;
        const newTimelineMonths = Math.ceil(amountRemaining / realisticMonthly);
        const newTargetDate = new Date();
        newTargetDate.setMonth(newTargetDate.getMonth() + newTimelineMonths);
        
        optimizations.goal_updates = {
          target_date: newTargetDate.toISOString()
        };
        optimizations.savings_strategy_updates = {
          monthly_contribution: realisticMonthly
        };
        
        console.log(`‚è∞ Adjusting timeline to be more realistic: ${newTimelineMonths} months`);
      }
    }
    
    // Apply optimizations if any were identified
    if (Object.keys(optimizations).length > 0) {
      optimizations.update_reason = "goal_optimization";
      optimizations.life_event_context = {
        trigger_event: "goal_refinement",
        impact_description: "Optimizing goal based on actual progress patterns"
      };
      
      const update = await awo.savings.updateGoal(goalId, optimizations);
      
      console.log('üéØ Goal optimized based on progress analysis!');
      console.log(`üìä Completion probability improved: ${progressAnalysis.completionProbability}% ‚Üí ${update.impact_analysis.completion_probability}`);
      
      return update;
    } else {
      console.log('‚úÖ Goal is well-optimized, no changes needed');
      return { message: "Goal is already well-optimized" };
    }
    
  } catch (error) {
    console.error('‚ùå Failed to optimize goal:', error.message);
    throw error;
  }
}

// Seasonal goal adjustment
async function adjustGoalForSeason(goalId, season, expectedIncomeChange) {
  try {
    const seasonalAdjustments = {
      summer: {
        description: "Summer season adjustment with potential overtime opportunities",
        contribution_multiplier: 1.1,
        boost_opportunities: ["overtime_pay", "seasonal_work"]
      },
      winter: {
        description: "Winter season adjustment with higher expenses",
        contribution_multiplier: 0.9,
        expense_increase: 500
      },
      holiday: {
        description: "Holiday season with bonus income and higher expenses",
        contribution_multiplier: 1.3,
        boost_opportunities: ["bonus_income", "tax_refunds"]
      },
      back_to_school: {
        description: "Back-to-school season adjustment",
        contribution_multiplier: 0.8,
        expense_increase: 1000
      }
    };
    
    const adjustment = seasonalAdjustments[season];
    if (!adjustment) {
      throw new Error(`Unknown season: ${season}`);
    }
    
    const currentGoal = await awo.savings.getGoal(goalId);
    const newContribution = Math.round(currentGoal.savings_plan.monthly_contribution * adjustment.contribution_multiplier);

        monthly_contribution: newContribution,
        boost_contributions: {
          ...currentGoal.savings_plan.boost_contributions,
          ...(adjustment.boost_opportunities?.reduce((acc, opportunity) => {
            acc[opportunity] = true;
            return acc;
          }, {}) || {})
        }
      },
      life_event_context: {
        trigger_event: "seasonal_adjustment",
        impact_description: adjustment.description,
        income_change: expectedIncomeChange ? {
          change_type: expectedIncomeChange > 0 ? "increase" : "decrease",
          temporary: true,
          seasonal: true
        } : undefined
      },
      smart_feature_updates: {
        seasonal_adjustments: true
      },
      update_reason: "seasonal_optimization"
    });
    
    console.log(`üåü Goal adjusted for ${season} season!`);

    return update;
  } catch (error) {

  }

    def seasonal_adjustment(self, goal_id: str, season: str, income_change_percentage: float = 0) -> Dict:
        """Adjust goal for seasonal income/expense changes"""
        seasonal_multipliers = {
            'holiday': 1.3,    # Holiday bonuses and gifts
            'summer': 1.1,     # Overtime opportunities
            'winter': 0.9,     # Higher heating costs
            'back_to_school': 0.8  # School expenses

        multiplier = seasonal_multipliers.get(season, 1.0)
        
        current_goal = self.get_goal(goal_id)
        current_contribution = current_goal['savings_plan']['monthly_contribution']
        
        # Apply seasonal adjustment
        base_adjustment = current_contribution * (multiplier - 1.0)
        income_adjustment = current_contribution * (income_change_percentage / 100)
        new_contribution = round(current_contribution + base_adjustment + income_adjustment)
        
        updates = {
            'savings_strategy_updates': {
                'monthly_contribution': new_contribution
            },
            'life_event_context': {
                'trigger_event': 'seasonal_adjustment',
                'impact_description': f'{season.title()} season adjustment with {multiplier}x factor'
            },
            'smart_feature_updates': {
                'seasonal_adjustments': True
            },
            'update_reason': 'seasonal_optimization'
        }
        
        try:
            result = self.update_goal(goal_id, updates)
