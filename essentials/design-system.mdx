# Design System

The AWO Design System provides a comprehensive set of design standards, components, and patterns that ensure consistency and quality across all AWO platform touchpoints.

## Design Philosophy

<CardGroup cols={2}>
  <Card title="Culturally Relevant" icon="heart">
    Designs that resonate with African women's experiences and cultural context
  </Card>
  <Card title="Accessible First" icon="universal-access">
    Inclusive design that works for users of all abilities and technology access levels
  </Card>
  <Card title="Trust & Transparency" icon="shield-check">
    Visual elements that build confidence in financial decisions and data security
  </Card>
  <Card title="Empowerment Focused" icon="trending-up">
    Designs that celebrate progress and encourage financial growth
  </Card>
</CardGroup>

## Brand Identity

### Brand Values

<Tabs>
  <Tab title="Visual Principles">
    **Empowerment Through Clarity**
    - Clean, uncluttered interfaces that reduce cognitive load
    - Clear information hierarchy that guides users naturally
    - Progressive disclosure to prevent overwhelm

    **Cultural Resonance**
    - Warm, welcoming color palette inspired by African landscapes
    - Visual metaphors that connect to local experiences
    - Imagery that represents the diversity of African women

    **Trust & Security**
    - Professional aesthetics that inspire confidence
    - Consistent branding that builds recognition
    - Transparent data visualization that promotes understanding

    **Growth & Progress**
    - Visual elements that celebrate achievements
    - Progress indicators that motivate continued engagement
    - Upward-trending visual language throughout the platform
  </Tab>
  
  <Tab title="Brand Personality">
    **Adjectives that Define AWO:**
    - **Empowering**: Gives users control and confidence
    - **Trustworthy**: Reliable and transparent in all interactions
    - **Culturally Aware**: Respectful and relevant to African contexts
    - **Progressive**: Forward-thinking while honoring tradition
    - **Supportive**: Encouraging and never judgmental
    - **Professional**: Serious about financial services while remaining approachable

    **Voice & Tone:**
    - **Encouraging**: "You're making great progress with your savings!"
    - **Educational**: "Here's what your DIVA score means and how to improve it"
    - **Celebratory**: "Congratulations on reaching your savings goal!"
    - **Supportive**: "We're here to help you achieve your financial dreams"
    - **Respectful**: Acknowledging cultural practices like Chama traditions
  </Tab>
  
  <Tab title="Logo System">
    **Primary Logo**
    ```
    [AWO Logo - Full Color]
    - Use on white or light backgrounds
    - Minimum size: 120px width
    - Clear space: Logo height on all sides
    ```

    **Logo Variations**
    ```
    [Logo Lockups]
    - Horizontal: Primary usage
    - Stacked: Square format applications
    - Icon Only: Small spaces, app icons
    - Monochrome: Single color applications
    ```

    **Usage Guidelines**
    - Never stretch or distort the logo
    - Maintain clear space requirements
    - Use provided color variants only
    - Don't place on busy backgrounds
    - Ensure adequate contrast ratios

    **File Formats**
    - **SVG**: Web and scalable applications
    - **PNG**: High-resolution with transparency
    - **JPEG**: Photography applications
    - **PDF**: Print applications
  </Tab>
</Tabs>

### Color Usage Guidelines

<AccordionGroup>
  <Accordion title="Accessibility Standards">
    **WCAG 2.1 AA Compliance**
    - Minimum contrast ratio of 4.5:1 for normal text
    - Minimum contrast ratio of 3:1 for large text (18px+ or 14px+ bold)
    - Color cannot be the only means of conveying information

    **Tested Color Combinations:**
    ```css
    /* High Contrast Combinations (AAA) */
    .text-on-primary { color: #FFFFFF; background-color: #E63946; } /* 5.8:1 */
    .text-on-secondary { color: #FFFFFF; background-color: #457B9D; } /* 4.9:1 */
    .text-on-success { color: #FFFFFF; background-color: #16A34A; } /* 6.2:1 */

    /* Standard Contrast Combinations (AA) */
    .primary-on-light { color: #E63946; background-color: #FFFFFF; } /* 5.8:1 */
    .secondary-on-light { color: #457B9D; background-color: #F9FAFB; } /* 4.7:1 */
    .text-primary { color: #111827; background-color: #FFFFFF; } /* 16.6:1 */
    ```
  </Accordion>

  <Accordion title="Color Psychology & Usage">
    **Primary Red (#E63946) - Confidence & Action**
    - Primary buttons and call-to-action elements
    - Important alerts and notifications
    - Progress indicators showing positive movement
    - Navigation active states

    **Secondary Blue (#457B9D) - Trust & Stability**
    - Secondary buttons and supportive actions
    - Financial data and DIVA score elements
    - Headers and navigation elements
    - Links and interactive elements

    **Accent Gold (#F1C40F) - Achievement & Wealth**
    - Achievement badges and celebrations
    - Premium features and upgrades
    - Success states and completed goals
    - Special offers and highlights

    **Semantic Colors - Communication**
    - **Success Green**: Completed transactions, achieved goals
    - **Warning Orange**: Attention needed, pending actions
    - **Error Red**: Failed operations, critical alerts
    - **Info Blue**: Educational content, helpful tips

### Typography Guidelines

<AccordionGroup>
  <Accordion title="Hierarchy & Spacing">
    **Visual Hierarchy Rules:**
    - Use font size, weight, and color to create clear hierarchy
    - Maintain consistent spacing between text elements
    - Limit to 3-4 font sizes per page for clarity
    - Use color and weight changes sparingly for emphasis

    **Spacing Guidelines:**
    ```css
    /* Vertical rhythm using consistent spacing scale */
    .text-spacing {
      margin-bottom: 1.5rem; /* 24px baseline */
    }

    .heading-spacing {
      margin-top: 2rem;     /* 32px before headings */
      margin-bottom: 1rem;   /* 16px after headings */
    }

    .paragraph-spacing {
      margin-bottom: 1rem;   /* 16px between paragraphs */
    }

    .list-spacing li {
      margin-bottom: 0.5rem; /* 8px between list items */
    }
    ```
  </Accordion>

  <Accordion title="Responsive Typography">
    ```css
    /* Responsive font scaling */
    @media (max-width: 768px) {
      :root {
        --text-5xl: 2.25rem;  /* 36px on mobile */
        --text-4xl: 1.875rem; /* 30px on mobile */
        --text-3xl: 1.5rem;   /* 24px on mobile */
      }
    }

    /* Fluid typography for smooth scaling */
    .heading-fluid {
      font-size: clamp(1.5rem, 4vw, 3rem);
    }

    .body-fluid {
      font-size: clamp(0.875rem, 2vw, 1.125rem);
    }

    /* Container-based responsive text */
    @container (max-width: 600px) {
      .card-title {
        font-size: var(--text-lg);
      }
    }
    ```
  </Accordion>
